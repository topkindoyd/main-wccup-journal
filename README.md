# main-wccup-journal

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态网页形式存放，不依赖特定域名或网站，仅作为内容存档与展示的载体。

仓库本身不提供搜索、交互或动态服务，仅保持页面文件的原始结构与可访问性。

## 目录说明

```
main-wccup-journal/
├── index.html          # 入口页面（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── example-01.html
│   ├── example-02.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下为各归档页面，文件名具备一定可读性，便于识别内容。
- `assets/` 存放页面引用的样式、脚本及图片等静态资源，保持相对路径引用。
- `index.html` 可作为导航或说明页，非必需，可根据需要保留或删除。

## 页面归档说明

- 每个 HTML 文件均为独立完整页面，可直接在浏览器中打开。
- 页面内不嵌入外部不可控资源（如远程广告、统计脚本等），保持自包含。
- 页面内容以信息展示为主，不包含表单提交、用户登录等交互功能。
- 归档页面可能定期更新或新增，旧页面一般保留，不做强制删除。
